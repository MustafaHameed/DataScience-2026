1 Introduction

This lab introduces Exploratory Data Analysis (EDA) using the built-in mtcars dataset in R.

1.1 Learning Objectives

  1. Understand the structure of a dataset
  2. Perform basic exploratory analysis
  3. Visualize data using histograms and scatter plots
  4. Apply filtering, subsetting, and aggregation

12 Subsetting Data

mt1 <- mtcars[mtcars$mpg > 20, c("gear", "mpg")]
View(mt1)

13 Aggregation: Mean MPG by Gear

mt2 <- aggregate(. ~ gear,
                 mtcars[mtcars$mpg > 20, c("gear", "mpg")],
                 mean)
View(mt2)

14 Post-Aggregation Filtering

mt3 <- subset(mt2, mpg > 25)
View(mt3)

15 Key Takeaways

Exploratory Data Analysis is the foundation of all data science and machine learning workflows.
